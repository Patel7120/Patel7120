## Architecture > Code

Most production issues aren't caused by a single service.

They're usually hiding somewhere between:

```text
Client → API → Service → Queue → Worker → Database → Another Service → Unexpected Behaviour
```

The further down the chain the issue is hiding, the more interesting it becomes.
